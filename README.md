# Space Travel

A Node App for the calculation of the fastest method of transport for fictitious space travel.

## Installation

Use git to clone

```bash
git clone https://github.com/kwenamokgohlwa/space_travel.git
cd space_travel
```

Use npm to install

```bash
npm install
```

Run Tests - NPM

```bash
npm test
```

Run App - Node

```bash
node src/main
```
